# 用于展示某模块下的组件，帮助调试或团队合作
> 在开发模式下预览组件使用

[!预览图片](./preview.png)

```sh
yarn add iwe7-books
```

```ts
import { BooksModule } from "iwe7-books";
@NgModule({
  imports: [BooksModule],
})
```

```html
<app-books [modules]="modules"></app-books>
```
